
# Browser-Toolkit

This is a web app that gives you all browser 
functionalites and more in one browser tab

## Features

- Searching web
    - Google
    - YouTube
    - StackoverFlow
    - Reddit
    - Dictionary
- Making notes
    - With title unlike others
    - Local storage
- Bookmarks
    - Easy to find boookmarks with titles
    - custom colors
- Clock
    - Digital Clock
    - Stopwatch
    - Timer
